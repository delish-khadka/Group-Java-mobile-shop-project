=================================ReadMe=======================================
______________________________________________________________________________

> Insert all the values in the empty fields.
> Select the catogory.
> Select the recommendation type.
> Insert price not less then or equal to 0.
> Select the quality of the accessory.
> Press Add button to put the value inside the table.
> Press Clear button to clear all the fields.
> Press Exit button to close the window.
> Press Delete button to delete the selected row from the table.

  To search the value inside the table, first insert price or select the
     category and hit "Search by price" or "Search by category" resp.
